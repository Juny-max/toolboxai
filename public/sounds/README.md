# Pac-Man Sound Files

To enable authentic Pac-Man sounds, place the following audio files in this directory:

## Required Sound Files:
- `waka.mp3` or `waka.wav` - The iconic waka-waka eating sound
- `power-pellet.mp3` or `power-pellet.wav` - Power pellet eating sound
- `eat-ghost.mp3` or `eat-ghost.wav` - Ghost eating sound
- `death.mp3` or `death.wav` - Pac-Man death sound
- `intro.mp3` or `intro.wav` - Game start/level start music
- `siren.mp3` or `siren.wav` - Background siren (optional)

## Where to get sounds:
You can find these sounds from:
1. Classic Arcade Sound Effects libraries (royalty-free)
2. Sound effect websites like freesound.org (with proper licensing)
3. Game asset stores

## Note:
The original Pac-Man sounds are copyrighted by Namco/Bandai Namco.
Please ensure you have proper licensing for any sounds you use.

If sound files are not present, the game will use synthesized fallback sounds.
