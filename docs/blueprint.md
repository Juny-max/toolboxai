# **App Name**: ToolboxAI

## Core Features:

- Text Case Converter: Convert text to various cases (Uppercase, Lowercase, Capitalize).
- JSON Formatter/Validator: Format and validate JSON code.
- Color Palette Generator: Generate a color palette based on user preferences or the primary color. Includes tools to choose/validate WCAG contrast with tool prompts
- Password Generator: Generate strong, random passwords based on user-specified criteria.
- QR Code Generator: Generate QR codes from user-provided URLs or text.
- Image Compressor: Compress images client-side to reduce file size using tool guidance.
- Code Assistant: Given a piece of code in a particular language, describe what the code does, identify potential errors, and/or suggest improvements, providing code suggestions as appropriate.

## Style Guidelines:

- Primary color: Soft neon purple (#BD93F9) for a modern, sleek look.
- Background color: Light gray (#F2F2F7) to provide a clean backdrop.
- Accent color: Electric blue (#8BE9FD) for interactive elements and highlights.
- Headline font: 'Space Grotesk', a sans-serif for headlines, paired with 'Inter', a sans-serif for body text. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use Lucide icons for all buttons and tool actions to ensure a consistent and modern look.
- Smooth transitions and subtle animations throughout the app using Framer Motion to enhance user experience.
